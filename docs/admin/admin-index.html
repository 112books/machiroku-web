1. Flux de treball diari

Per publicar qualsevol canvi al web, el flux és sempre el mateix:

cd ~/Documents/Obsidian/machiroku_hugo
./symc-machiroku.sh
→ Opció 4 (Deploy)
⚠️ Sempre opció 4 (Deploy) per publicar. L'opció 3 (Push) només guarda el codi sense reconstruir el web.

2. Editar menús del migdia

Fitxer: data/ca/menus.yaml (i els equivalents data/es/ i data/en/)

Canviar un preu

- id: "menu-sushi"
  preu: 17.00   ← canviar aquí

Desactivar un menú temporalment

actiu: false   ← no apareix al web, però es conserva

Canviar els plats inclosos

plats:
  - "Sopa miso"
  - "Nou plat aquí"   ← afegir o modificar
  - "Te verd"

3. Editar la carta nocturna

Fitxer: data/ca/plats.yaml (i els equivalents ES i EN)

Afegir un plat nou

- id: "nom-unic"          ← sense espais ni accents
  nom: "Nom del plat"
  categoria: "entrants"   ← veure taula de categories
  ordre: 5               ← ordre dins la categoria
  actiu: true
  destacat: false        ← true = badge "Destacat"
  preu: 6.50
  foto: ""               ← deixar buit fins tenir foto
  descripcio: "Descripció breu del plat"
  alergics:
    - "soja"             ← veure taula d'al·lèrgens

5. Avisos temporals

Fitxer: data/avisos.yaml

Per activar un avís de tancament o especial:

- id: "tancat-setmana-santa"
  actiu: true              ← canviar a true per activar
  des_de: "2026-04-01"    ← data inici (YYYY-MM-DD)
  fins_a: "2026-04-06"    ← data fi (YYYY-MM-DD)
  prioritat: "alta"       ← alta (vermell) | mitjana (groc) | baixa (verd)
  tipus: "banner"         ← banner (barra) | popup (modal)
  titol:
    ca: "Tancats per Setmana Santa"
    es: "Cerrados por Semana Santa"
    en: "Closed for Easter"
  text:
    ca: "Tornem el dimarts 7 d'abril."
    es: "Volvemos el martes 7 de abril."
    en: "We reopen on Tuesday 7 April."
  cta:
    ca: ""
    es: ""
    en: ""
  link: ""
✓ Quan passin les dates, l'avís desapareix sol. No cal desactivar-lo manualment.

6. Fotos de plats

  • Mida recomanada: 800×600 px (proporció 4:3)
  • Format: .jpg o .webp
  • Pes màxim: 200 KB
  • Carpeta: static/images/menus/ o static/images/plats/
foto: "/images/plats/nom-del-fitxer.jpg"
✓ Si la foto no existeix, la card es mostra igualment sense imatge — no dona error.
